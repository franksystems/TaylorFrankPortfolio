# Taylor Frank Portfolio

A modern Next.js portfolio site designed for staff-level software engineering positioning.

## Stack
- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. Import the repository into Vercel.
3. Vercel should auto-detect Next.js with no extra configuration.
4. Add your custom domain in Vercel settings.
